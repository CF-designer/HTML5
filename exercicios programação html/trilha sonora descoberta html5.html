Estrutura Html5:

Como é a Estrutura de um Documento HTML?

A estrutura básica de um documento HTML é composta por três partes: A declaração do tipo de Documento, o Cabeçalho e o Corpo.

Declaração do Tipo de Documento

  • A instrução <!DOCTYPE html> deve ser a primeira linha do documento.
  • Esta instrução informa ao navegador que o código é HTML.

    • Cabeçalho

  • Delimitado pelas tags <HEAD> e </HEAD>.
  • Contém informações sobre o documento, como o título, a codificação e links para arquivos CSS e JavaScript.

    • Corpo

  • Delimitado pelas tags <BODY> e </BODY>.
  • Contém conteúdo da página, como textos, imagens, formulários, vídeos, mapas, entre outros.
    • As tags HTML, são formadas por uma tag de abertura, o conteúdo e uma tag de fechamento.A tag de abertura começa com o símbolo <> e a tag de fechamento termina com o símbolo </>

    O que são as TAGS, como identifica-las e como usa-las?

    TAGS HTML são instruções que informam ao navegador como formatar o texto. Você pode usar tags para definir itálico, quebras de linha, objetos, bullet points e muito mais. Essas tags estão presentes no HTML (Hyperyext Markup Language) de todas as páginas da internet. Ao todo são mais de 100 tags que podem ser utilizadas!!

    Exemplos de tags HTML iniciais mais comuns:

  • <p></p> – Principal tag de texto, compõe um parágrafo;
  • <span></span> – Apesar de ter uma funcionalidade e características parecidas com os parágrafos, costumam ser utilizadas apenas para pequenas informações, como legendas de um formulário, legendas de uma imagem, etc. Também pode ser utilizada para formar um container;
  • <pre></pre> – Tag utilizada para representar texto pré-formatado. Muito utilizada para inserir códigos;
  • <b></b> – Transforma o conteúdo em negrito;
  • <i></i>– Transforma o conteúdo em itálico;
  • <br/> – Essa tag não necessita de fechamento, ela executa a função de quebra de linha.
  • <hr/>– Essa tag não necessita de fechamento, ela forma uma linha horizontal.
